# MDE para predição de reprovação na EaD


Este estudo preliminar buscou testar diferentes modelos preditivos com vistas à detecção antecipada de alunos em risco de reprovação nas disciplinas em oferta.
A partir de dados coletados no Ambiente Virtual de Aprendizagem utilizado por um curso superior na modalidade de edudação a distância em uma IES pública brasileira integrante da Sistemática UAB (Universidade Aberta do BRasil), foram realizados diversos procedimentos de pré-processamento e implementados 6 diferentes modelos preditivos. Os resultados foram apresentados e analisados.

Neste repositório encontram-se os códigos-fonte do artigo final e do script desenvolvido como requisito parcial para aprovação na disciplina PCI410068 Machine Learning e Deep Learning ministrada pelo Prof. Dr. Gustavo Medeiros de Araujo.
